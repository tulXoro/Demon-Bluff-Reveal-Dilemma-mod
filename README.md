# Demon Bluff - Reveal Dilemma

## Description

This is a [Demon Bluff](https://store.steampowered.com/app/3522600/Demon_Bluff/) mod which hopes to encourage players to think strategically when revealing cards.

## New Roles

### Auditor

Good Villager. Pick two unrevealed characters. Heal 2 for each non-Auditor villager you selected.

### Saboteur

Good Outcast. Deals **4** damage to the player when revealed. Disguises as a random Villager.

### Shroud

Evil Demon. Deals 1 damage everytime the player reveals a card while it is alive.

### Martyr (WIP)

Evil Minion.

## Installation

### ***Prerequistes***

You will need to download [Melon Loader](https://melonwiki.xyz/#/).
You will also need to game installed, of course.

### Directions

1. Open Melon Loader. Select/search "Demon Bluff Playtest".
2. Enable Nightly builds and install.
3. Launch the game first without any mods.
4. Navigate to the game directory and you'll see the `mods` folder. Drag and drop this mod's `.dll` file in there and run the game.
5. Enjoy the game.

**Note:** This game does not add any sprites, so you may want to add your own.
